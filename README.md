# Portfolio v2

An interactive 3D portfolio website built with Three.js.

## Features

- 3D room environment
- Interactive objects that open modals with portfolio content
- Responsive design
- Mouse controls for camera movement
- GitHub Pages deployment ready

## Interactive Elements

- **Person on Sofa**: Click to view About section
- **Bookshelf**: Click to view Skills section  
- **PC Setup**: Click to view Works section
- **Smartphone**: Click to view Contact section

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run deploy
```

## Tech Stack

- Three.js for 3D graphics
- Vite for build tooling
- Vanilla JavaScript
- GitHub Pages for hosting
